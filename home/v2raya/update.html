更新

通过脚本更新

脚本会自动获取最新版本然后安装，因此，你只需要重新运行一次安装脚本就能更新核心

通过软件源更新

如果你从软件源安装了 v2rayA，那么在你更新系统与软件的时候，v2rayA 的更新也会随之安装。以 Debian 系统为例，更新系统与软件的命令为：

sudo apt update
sudo apt upgrade

Gnome Software、Discover、系统更新器等图形化工具也可以更新 v2rayA。

手动更新

从 GitHub 的 Releases 下载新版本，然后覆盖安装即可。
